# Hinagata
WPF 学習用に作成したソリューションです。


## WithCommunityToolkit
最終的なひな型というか、アプリ開発時に参照できるものです。
カウンタアプリを MVVM アーキテクチャに従い、コミュニティツールキットとコンテンツコントロールを利用し実装しています。


## Hinagata.Test
Json 入出力のクラス作成時に単体テストをするために作成したものです。
xUnit を利用した簡易なテストを実施しています。

## Hinagata
ひな型として作成しようとしたものです。
コミュニティツールキットの存在もあり、学習用に段階的な実装をなるべく残した結果、ひな型と呼ぶには違和感があるものになりました。
親ウィンドウと、MVVM 構成で構成したウィンドウ、さらにコンテンツコントロールで実装したウィンドウがあります。
中を見ると、それぞれのコードを比較できます。
